# Wanderlust
A Full Stack Web Development Project built to explore, list, and manage travel destinations. Wanderlust combines a robust backend with a clean, user-friendly frontend to deliver a seamless experience for travelers and developers alike.

Features
-Add, edit, and delete travel listings

-User-friendly interface with EJS views

-MongoDB integration for persistent storage

-Error handling and validation for robust data integrity

-Modular project structure for maintainability

Tech Stack
Frontend: EJS, CSS

Backend: Node.js, Express.js

Database: MongoDB (Atlas)

Other: Middleware for validation, cloud configuration

git clone https://github.com/Mittal-Divyanshu/Wanderlust.git
